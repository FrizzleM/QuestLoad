Quest Web APK Installer

Step 1: connect your Quest. Step 2: pick an APK (apps) or a bundle folder (games with OBB). Step 3: install.
Chrome/Edge only (WebUSB).
1. Connect your Quest
Plug in headset, enable Developer Mode, then accept the USB debugging prompt in the headset.
2. Choose what to install
APK only installs apps. Bundle folder installs games: APK + OBB using the release.manifest metadata.
Choose an .apk (or drop it here) Must be an .apk, not a folder
APK
Choose a game bundle folder Must contain: release.manifest + APK + OBB (main.*.obb).
FOLDER
3. Install
APK installs immediately. Bundle installs APK + pushes OBB to /sdcard/Android/obb/<package>/.
